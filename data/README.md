# Data Folder

Place the datasets locally using the following structure:

```text
data/
├── bigdata2024classification/
│   ├── train.csv
│   └── test_without_labels.csv
│
├── pickle-files/
│   ├── processed_data.pkl
│   └── processed_test_data.pkl
│
└── timeseriesexercise3/
    └── dtw_test.csv
```

The actual dataset files are ignored by Git and should not be uploaded to GitHub.

The original notebook was executed partly in a Kaggle-style environment, so local paths may need to be adjusted before running.
